# Ephemeral Rooms Mobile App

React Native mobile app built with Expo.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Update API URL in `config/api.js` with your computer's IP address

3. Start Expo:
```bash
npx expo start
```

4. Scan QR code with Expo Go app (iOS/Android)

## Features

- Login/Signup
- Create/Join rooms
- Real-time chat
- Image sharing from camera/gallery
- Live location tracking
- Room management
